# EZDine

Restaurant Billing, POS, Order Management, Kitchen Display, Inventory, and BI SaaS platform.

## Monorepo Layout
- `apps/web` - Next.js web app (TypeScript + Tailwind)
- `apps/mobile` - Flutter app (tablet/mobile)
- `packages/shared` - Shared types, constants, and utilities
- `docs` - Architecture, contributing, and planning docs

## Quick Start (later)
This repo is scaffolded first. Installation and run steps will be added after dependencies are introduced.

## Principles
- Offline-first
- Fast (<1s interactions)
- Multi-tenant, multi-branch
- Role-based security
- Touch-friendly UX
